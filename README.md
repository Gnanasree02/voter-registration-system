
# 🗳️ Voter Registration System

A simple web app to register voters and manage voter data easily and securely.

---

## 💻 Tech Used

- 🐍 Python (Flask)  
- 🐬 MySQL  
- 🌐 HTML, CSS, JS  

---

## ✨ Features

- ✅ Voter registration form  
- ✅ Stores data in MySQL  
- ✅ Admin can view voter list  
- ✅ Validations (like age, ID)  

---

## 🚀 How to Run

1. 📥 Clone the project  
   ```bash
   git clone https://github.com/your-username/voter-registration-system.git
````

2. 🛠️ Install required packages

   ```bash
   pip install -r requirements.txt
   ```

3. 🗄️ Set up MySQL and update credentials in `app.py`

4. ▶️ Run the app

   ```bash
   python app.py
   ```

5. 🌐 Open in browser
   `http://localhost:5000`
